# Medical ChatBot

Live demo hosted on AWS: http://3.80.193.51:8080

## Overview
An LLM-powered medical assistant built with:
- Python
- Flask
- LangChain & LangGraph
- RAG with vector search
- AWS deployment

## Setup
1. Clone
2. Create virtual environment
3. Add `.env` file with keys
4. Install requirements
5. Run Flask app

⚠️ **No API keys or credentials are included in this repo.**
Use environment variables to configure them.

## Tech Stack
- Backend LLM orchestration
- Retrieval-Augmented Generation
- Vector DB support
- AWS hosting
